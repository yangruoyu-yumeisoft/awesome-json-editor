## 🌟 Why Awesome JSON Editor?

> Tired of complicated JSON tools? Try the simplest yet most powerful solution!

**Awesome JSON Editor** is a modern JSON editor designed for developers. It transforms tedious data processing into a visual feast. Whether you're a frontend engineer, backend developer, or data analyst, this tool can boost your productivity by **300%**!

### 💡 Core Advantages

- 🎯 **Single HTML** - Just one file, no installation needed
- ⚡ **Lightning Fast** - Runs locally, data never leaves your device
- 🎨 **Beautiful UI** - Dark theme + gradient design, easy on the eyes
- 📱 **Cross-Platform** - Perfect for PC, MAC, and Mobile

---

## ✨ Features

### 🌳 Dual View Modes

| Mode | Description | Use Case |
|------|-------------|----------|
| **Tree View** | Hierarchical foldable browsing | Complex nested data |
| **Mind Map** | Radial visualization | Data structure analysis |

### 🛠️ Powerful Editing

- ✅ **Real-time Formatting** - One-click JSON beautification
- ✅ **Syntax Validation** - Instant error detection with precise location
- ✅ **Undo/Redo** - Complete operation history
- ✅ **Multi-Tabs** - Handle multiple JSON files simultaneously
- ✅ **Double-click Rename** - Quickly rename tabs

### 🔄 Smart Code Generation

Select any JSON node and generate with one click:

| Feature | Output |
|---------|--------|
| ☕ **Java Object** | Lombok-annotated POJO class |
| 🗄️ **MySQL Schema** | Complete CREATE TABLE statement |
| ➕ **INSERT Statement** | Batch insert SQL |
| 📝 **UPDATE Statement** | Update SQL template |

### ⌨️ Efficient Shortcuts

```
↑ / ↓        Navigate up/down
← / →        Expand/collapse node
Enter        Add child/insert below
Shift+Enter  Insert above
Tab          Insert below
F2           Edit key/value
[ / ]        Switch data type
Ctrl+[ / ]   Move node up/down
Delete       Delete node
Ctrl+Z       Undo
Ctrl+Y       Redo
```

## 🚀 Quick Start

### Option 1: Direct Open (Recommended)

```bash
# Clone the repo
git clone https://github.com/yangruoyu-yumeisoft/awesome-json-editor.git

# Enter directory
cd awesome-json-editor

# Open in browser
open index.html
```

### Option 2: Local Server

```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .

# PHP
php -S localhost:8080
```

Then visit `http://localhost:8080`
